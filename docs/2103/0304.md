# [Ubuntu] GPU 모델명과 벤더사 찾기
먼저 PCI ID를 업데이트 해온다(sudo 필요)
```
sudo update-pciids
```
그 다음 lspci 나 lshw 명령어를 이용해서 모델명을 조회할 수 있다.
```
- Option 1
lspci | grep VGA

- Option 2
sudo lshw -C display
(sudo가 없는 경우 Access Denied가 발생)
```
